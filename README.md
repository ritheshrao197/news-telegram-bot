# 📰 Telegram News Bot

A powerful Telegram bot that fetches news from multiple sources and provides detailed summaries with at least 50 words per article.

## ✨ Features

- 🇮🇳 India, 🌍 World, and 💻 Technology news
- 📝 Minimum 50-word detailed summaries
- 🔄 Automatic hourly news updates
- 💾 Database storage for fast retrieval
- 📊 News statistics and analytics
- 🏘️ District/Taluk level local news support
- ➕ Custom RSS feed addition

## 🚀 Quick Start

### Prerequisites

- Python 3.8+
- Telegram Bot Token (from [@BotFather](https://t.me/botfather))

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/YOUR_USERNAME/news-telegram-bot.git
cd news-telegram-bot
```
